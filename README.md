# Usage
* Create directory named `music`
* `a` - edit artist
* `t` - edit title
* `f` - edit featuring
* `m` - edit mod
* `<enter>` - next song
* `q` - quit

# Compatibility
There are some unresolved issues with taglib2, which is default in Arch. Linking with taglib1 works, did not dig any further.
